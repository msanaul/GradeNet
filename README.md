<h1>🎓 GradeNet</h1>

<p><strong>A student-teacher grade management system built using Java (Android) following MVC architecture.</strong></p>

<h2>📌 Overview</h2>
<p>
GradeNet is designed to streamline academic grade management between students and teachers.
It provides a structured system for managing, viewing, and updating academic performance.
</p>
<ul>
  <li>👨‍🏫 Teachers can upload and manage grades</li>
  <li>👨‍🎓 Students can view their performance</li>
</ul>

<h2>🧩 Features</h2>
<ul>
  <li>🔐 Role-based access (Student / Teacher)</li>
  <li>📝 Grade management system</li>
  <li>📊 Performance tracking</li>
  <li>🗂 Structured database handling</li>
  <li>🏗 MVC architecture implementation</li>
</ul>

<h2>🛠 Tech Stack</h2>
<ul>
  <li>Java</li>
  <li>Android</li>
  <li>MVC Architecture</li>
  <li>SQLite (or relational database)</li>
  <li>Gradle</li>
</ul>

<h2>🧠 System Design</h2>
<ul>
  <li><strong>Model:</strong> Handles data logic and database operations</li>
  <li><strong>View:</strong> Android UI components</li>
  <li><strong>Controller:</strong> Manages logic and connects Model & View</li>
</ul>

<h2>🚀 Getting Started</h2>

<h3>Prerequisites</h3>
<ul>
  <li>Android Studio</li>
  <li>Java SDK</li>
</ul>

<h3>Installation</h3>
<pre>
git clone https://github.com/msanaul/GradeNet.git
cd GradeNet
</pre>

<ol>
  <li>Open the project in Android Studio</li>
  <li>Build using Gradle</li>
  <li>Run on emulator or physical device</li>
</ol>
